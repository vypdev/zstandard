## test

This package does not run unit tests in this directory. Web behaviour is covered by **integration tests** in the example app.

**How to run web tests:** from the repo root run `./scripts/test_web_integration.sh` (requires Chrome and ChromeDriver on port 4444). It runs the integration tests in `example/integration_test/`.

See `example/README.md` and `docs/development/emulator-setup.md` (Web section) for more info.
